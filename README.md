### 🔍 "기업 분석, 이제 자동으로! Catching(캣칭)"

**✨ 더 이상 회사 조사에 시간을 낭비하지 마세요!**  
채용 공고를 볼 때마다 기업 검색하고, 많은 정보를 정리하느라 지친 적 있나요?  
Catching은 원하는 직무의 기업 분석에서 내용 정리까지 모두 자동으로 한 번에 제공하는  
나만의 리서치 비서로 지원 준비를 빠르게·깊게·정확하게 만들어주는 확장프로그램입니다.

**🎯 이런 분들을 위한 기업 조사 자동화 툴이에요**  
기업 분석이 너무 번거롭고 시간이 오래 걸리는 취준생  
채용 공고를 보며 “이 회사 어떤 회사지?”를 매번 검색하는 분  
AI 기반으로 빠르게 회사 관련 정보를 받고 싶은 분  
조사한 내용까지 노션에 자동으로 기록하고 싶은 분

**🐱 Catching만의 강력한 기능**  
웹에서 채용 공고를 탐색하면 회사명 자동 추출  
직무를 캡쳐하면 OCR로 텍스트 자동 변환  
회사·직무 기반 맞춤 AI 분석 결과 제공  
결과를 버튼 한 번으로 노션에 기록  
많이 검색한 회사+직무 Top5 차트 및 결과 제공

**👨🏻‍💻 이렇게 사용해보세요 !**  
1️⃣ 확장프로그램을 설치하고 간편하게 구글 로그인을 해보세요  
2️⃣ 채용 사이트를 탐색하면 회사명이 자동으로 추출돼요  
3️⃣ 분석하고 싶은 직무 부분을 캡쳐하면 자동으로 입력해줘요  
4️⃣ 회사 + 직무 기반으로 AI가 분석 결과를 자동 생성해줘요  
5️⃣ 마음에 드는 정보는 노션에 자동 기록하거나 PDF로 저장해보세요

</blockquote>

## ⌨️ 주요 기능

1. 회사명 자동 추출 : 채용 사이트에서 공고를 열면, 페이지 내에서 회사 이름을 자동으로 식별하고 추출합니다.
2. 직무 영역 캡쳐 + OCR 텍스트 추출 : 확장프로그램에서 제공하는 캡쳐 기능으로 직무를 선택하면 OCR을 통해 텍스트로 변환합니다.
3. 회사 + 직무 기반 AI 분석 : 회사명과 직무를 이용해 AI 분석 결과를 받아옵니다.
4. 노션 자동 기록 : AI 분석 결과를 Notion API를 통해 사용자 DB 페이지에 자동으로 기록할 수 있습니다.
